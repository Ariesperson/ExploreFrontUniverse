# 从输入URL到页面加载的全过程

1.输入UIL确认跳转

2.查找缓存：浏览器缓存-系统缓存-路由器缓存

3.DNS域名解析：解析域名实际的IP地址，DNS服务器是基于UDP的，因此会用到UDP协议。

4.建立TCP连接（TCP三次握手）

5.发送HTTP或者HTTPS请求（请求资源）

6.服务器相应请求并返回结果（返回html文件）

7.关闭TCP连接（TCP四次挥手）

8.浏览器渲染页面

​	构建Dom树

​    构建CSS树

​	构建render树

​	布局

​	绘制

9.JS引擎执行JS代码

​	创建windows对象

​	加载文件

​	预编译

​	解释执行
